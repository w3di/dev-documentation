## База данных и СУБД

База данных — набор данных в структурированном виде

СУБД — система для доступа к данным, их создания, изменения и удаления

## Типы баз данных

Реляционные БД:
Данные хранятся в таблицах со строками и столбцами

Особенности:
- Строгая типизация данных
- Одно значение на пересечении строки и столбца
- Столбцы имеют имя и тип
- Порядок столбцов фиксирован при создании
- Таблица может быть пустой, но должна иметь хотя бы один столбец
- Результат запроса — таблица

key-value БД:
Данные хранятся как пары ключ-значение, где ключ — уникальный идентификатор

Документоориентированные БД:
Данные хранятся в виде документов, похожих на JSON

## Структура таблицы реляционных БД

Таблица состоит из:
- строк (записи)
- столбцов (поля, атрибуты)

## Первичный ключ

Поле или набор полей, однозначно определяющий запись в таблице

Необязателен, целостность может обеспечиваться на уровне приложения

## Внешний ключ

Поле или набор полей в одной таблице, ссылающееся на первичный ключ другой таблицы

Таблица с внешним ключом — дочерняя, таблица с первичным ключом — родительская
Внешний ключ гарантирует, что значение существует в родительской таблице

Необязателен
