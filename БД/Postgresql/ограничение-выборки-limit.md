## Ограничение выборки, оператор LIMIT

LIMIT позволяет извлечь определённый диапазон записей из одной или нескольких таблиц.

```sql
SELECT поля_выборки
FROM список_таблиц
LIMIT количество_записей_для_вывода [OFFSET количество_пропущенных_записей];
```
Общая структура запроса с LIMIT

Если не указать OFFSET, отсчёт будет вестись с начала таблицы.

```sql
SELECT * FROM Company LIMIT 3 OFFSET 2;
```
Пропускает первые 2 строки, затем выводит следующие 3 записи

В данном запросе происходит пропуск первых двух строк таблицы, после чего выводятся следующие три записи.
