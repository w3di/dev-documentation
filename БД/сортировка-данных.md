## Сортировка, оператор ORDER BY

По умолчанию строки возвращаются в неопределённом порядке. ORDER BY упорядочивает записи.

```sql
SELECT поля_таблиц FROM наименование_таблицы
WHERE ...
ORDER BY столбец_1 [ASC | DESC][, столбец_n [ASC | DESC]]
```
Общая структура запроса с ORDER BY

Направление сортировки:
- ASC — по возрастанию (по умолчанию)
- DESC — по убыванию

```sql
SELECT name FROM Company ORDER BY name;
```
Выводит названия компаний в алфавитном порядке

Сортировка по типам данных:

Строковый тип:
- ASC: от A до Z (от А до Я)
- DESC: от Z до A (от Я до А)

Числовой тип:
- ASC: от меньшего к большему
- DESC: от большего к меньшему

Дата и время:
- ASC: от более ранней к более поздней
- DESC: от более поздней к более ранней

Булевый тип:
- ASC: false перед true
- DESC: true перед false

NULL значения:
- ASC: NULL последними
- DESC: NULL первыми

Сортировка по нескольким столбцам:

Столбцы указываются через запятую. Сначала сортировка по первому столбцу, при совпадении значений — по второму и т.д.

Правило сортировки применяется только к тому столбцу, за которым оно следует.

```sql
SELECT DISTINCT town_from, town_to FROM Trip
ORDER BY town_from, town_to DESC;
```
Сортирует по town_from по возрастанию, при совпадении — по town_to по убыванию
