## Числовой тип данных в SQL

Создание числовых данных в SQL: можно ввести число как литерал, получить из столбца таблицы или сгенерировать путём вычислений.

При вычислениях можно использовать стандартные арифметические операции (+, -, *, /) и изменять приоритеты вычислений с помощью скобок.

```sql
SELECT 2 * ((22 - 16) / (2 + 1)) AS calc_example;
```
Пример арифметических вычислений

## Математические функции

Для математических вычислений в SQL есть встроенные числовые функции:

- POWER(num, power) — вычисляет число в указанной степени
- SQRT(num) — вычисляет квадратный корень числа
- LOG(base, num) — вычисляет логарифм числа по указанному основанию
- EXP(num) — вычисляет e^num
- SIN(num) — вычисляет синус числа
- COS(num) — вычисляет косинус числа
- TAN(num) — вычисляет тангенс числа

## Округление чисел

Для округления числовых данных в SQL предусмотрены функции: CEIL, FLOOR, ROUND, TRUNC.

## Функции CEIL и FLOOR

CEIL округляет число к ближайшему целому числу в большую сторону, FLOOR — в меньшую.

```sql
SELECT CEIL(69.69) AS ceiling, FLOOR(69.69) AS floor;
```
CEIL возвращает 70, FLOOR возвращает 69

## Функция ROUND

ROUND округляет число к ближайшему целому. Если десятичная часть >= 0.5, округляет в большую сторону, иначе в меньшую.

```sql
SELECT ROUND(69.499), ROUND(69.5), ROUND(69.501);
```
ROUND(69.499) = 69, ROUND(69.5) = 70, ROUND(69.501) = 70

ROUND также позволяет округлять число до указанного количества знаков после запятой.

```sql
SELECT ROUND(69.7171,1), ROUND(69.7171,2), ROUND(69.7171,3);
```
Округление до 1, 2 и 3 знаков после запятой

Второй аргумент ROUND может принимать отрицательные значения. В этом случае цифры слева от десятичной точки становятся равными нулю на указанное количество, а дробная часть обрезается.

```sql
SELECT ROUND(1691.7,-1), ROUND(1691.7,-2), ROUND(1691.7,-3);
```
ROUND(1691.7,-1) = 1690, ROUND(1691.7,-2) = 1700, ROUND(1691.7,-3) = 2000

## Функция TRUNC

TRUNC аналогична ROUND, но вместо округления просто отбрасывает ненужные цифры. Также может принимать второй необязательный параметр.

```sql
SELECT TRUNC(69.7979,1), TRUNC(69.7979,2), TRUNC(69.7979,3);
```
TRUNC(69.7979,1) = 69.7, TRUNC(69.7979,2) = 69.79, TRUNC(69.7979,3) = 69.797

```sql
SELECT TRUNC(69.7979, -1);
```
TRUNC(69.7979, -1) = 60

## Работа со знаковыми числами

При работе с числовыми данными, в которых возможно наличие отрицательных значений, могут быть полезными функции SIGN и ABS.

## Функция SIGN

SIGN возвращает значение -1, если число отрицательное, 0, если число нулевое и 1, если число положительное.

```sql
SELECT SIGN(-69), SIGN(0), SIGN(69);
```
SIGN(-69) = -1, SIGN(0) = 0, SIGN(69) = 1

## Функция ABS

ABS возвращает абсолютное значение числа.

```sql
SELECT ABS(-69), ABS(0), ABS(69);
```
ABS(-69) = 69, ABS(0) = 0, ABS(69) = 69
